Nedir?

Nebim V3 Entegratör V3 kullanan müşterilerin kendisinin geliştirdiği veya çözüm ortaklarına yaptırmış olduğu uygulamaların V3 ile entegre olmasını sağlamak amacıyla geliştirilmiş bir araçtır.

Teknik Bilgiler

V3 Entegratör .Net Framework 4.0 ile geliştirilmiş bir ASP.Net MVC Web API uygulamasıdır. Internet Information Services (IIS) sunucusu üzerinde barındırılarak çalıştırılır.

Kurulum

V3 Entegratör uygulamasının kurulumu bir web sitesinin sunucunu üzerinde barındırılması gibidir.Yapılması gerekenleri aşağıdaki gibi listeleyebiliriz.

  1. Web Sunucunuzda IIS kurulu değilse öncelikle IIS Kurulumunu yapınız
  2. IIS kurulumu tamamlandıktan sonra IIS Manager uygulamasını çalıştırınız (bkz.Control Panel>Administrative Tools>Internet Information Services (IIS) Manager)
  3. IIS Manager uygulamasında Siteler bölümünü seçiniz.Ardından farenizle sağ tıklayıp Yeni Site Ekle işlemini başlatınız.
  4. Gelen ekranda uygulmanızın bulunduğu klasörü gösterip gerekli yetki ayarlarını yaptıktan sonra artık V3 Entegratör uygulaması kullanım için hazırdır
